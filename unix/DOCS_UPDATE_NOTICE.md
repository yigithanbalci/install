# Documentation Update Notice

## ⚠️ IMPORTANT: Configuration File Change

**As of the latest update, the configuration system has been migrated from `config.zsh` to `config.sh` for POSIX compliance.**

### What This Means:

- **New installations**: Use `config.sh` (POSIX bash)
- **Existing users**: Migrate from `config.zsh` to `config.sh`
- **All references to `config.zsh` in these docs should be read as `config.sh`**

### Why This Change?

- ✅ Works on all Unix-like systems (no zsh required)
- ✅ POSIX compliant and portable
- ✅ Works on bare minimum OS installations
- ✅ Better compatibility across different systems

### Migration:

See [MIGRATION_TO_POSIX.md](../MIGRATION_TO_POSIX.md) for detailed migration instructions.
